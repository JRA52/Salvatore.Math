Salvatore es una aplicación desarrollada con el fin de proporcionar al usuario una guía con ejemplos y ejercicios para ayudar a estudiar los temas de Factoriales y Coeficientes Binomiales 


Instrucciones de uso: 
Una vez ejecutado el programa, debemos de dirigirnos a cualquiera de los apartados
En dos de ellos únicamente se muestra teoría y algunos ejemplos
En los otros 2 encontramos unas guías interactivas para ambos temas respectivamente 


CoefBinomialGuia.xaml
Encontramos ejercicios propuestos en donde podemos ingresar el resultado y el programa corroborará que es el resultado correcto 
Adicionalmente tenemos la opción de ver la solución del proyecto por si se quiere corroborar el procedimiento mediante el apartado ver solución
